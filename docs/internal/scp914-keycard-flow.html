SCP-914 Keycard Flow

Scp914Changes.cs / Scp914Registry の現在実装を読んでまとめた概要表。O5 は SCPCB の OMNI 的な最終ギャンブル枠で、低位カードからも極低確率で到達可能だが、基本は壁を越えて MasterCard / Site Director まで進めてから狙う。

メモ・前提

共通 O5 ロール 全ての入力に対して、処理の最初に 0.2% で O5 に変換される超低確率ロールが走る（床ピックアップ/インベントリ共通）。これは「どのカードにも一応夢はある」ための最低保証で、通常の育成ルートとは別枠。
研究・管理系階層 Janitor < Scientist < Zone Manager < Research Coordinator < Containment Engineer < Facility Manager < Site Director < O5 の順で上がっていく。Facility Manager は C3-Admin3 なので Containment Engineer より明確に上位。
軍・武器系階層 Armory 1 < Guard = Chaos Intruder < Armory 2 < Security Chief < MTF Private < MTF Operative < MTF Captain = Chaos Insurgency Device < O5 のイメージ。
O5 / OMNI 的立ち位置 Site Director より上の最終カード。低位からの O5 直行は絶望的で、Research Coordinator / Armory 3 / Facility Manager 付近に Fine の壁がある。安定 Fine 連打では止まりやすく、Coarse / Rough で別系統へ戻すか、Very Fine / MasterCard で上振れを狙う設計。
O5 到達の実質ルート 現実的には Site Director の Very Fine が O5 50% / Destroy 50%、MasterCard の Very Fine が O5 10% / Site Director 25% / Facility Manager 15% / Janitor 10% / Destroy 40%。下位カードは MasterCard や Site Director へ近づくための足場。
O5 確率調整 個別の O5 抽選は Scp914Changes.cs の O5ProbabilityScale でまとめて調整する。共通 O5 ロールだけは O5WildcardChance で独立管理。
Administration 表記 Checkpoints + Intercom = Administration 1、ExitGates = Administration 2、AlphaWarhead = Administration 3。コード上の flags は KeycardAccessLevels でこの表記へ変換する。

大まかなフローチャート

研究 / 管理ルート

Janitor→ Scientist→ Zone Manager→ Research Coord.→ Containment Eng.→ Facility Manager→ Site Director→ O5

Research Coordinator 以降は Fine だけでは頭打ち。Coarse / Rough で別系統へ戻すか、Very Fine / MasterCard で壁を越える。

軍 / 武器アクセスルート

Armory 1→ Guard⇄ Chaos Intruder→ Armory 2→ Security Chief→ MTF Private→ MTF Operative→ Armory 3→ MTF Captain / Chaos Device→ Site Director→ O5

Armory 3 は Fine だと同帯ローテーション。Captain / Chaos Device へ抜けるには Very Fine を使う。

Chaos / ギャンブルルート

Quarter⇄ MasterCard⇄ PlayingCard→ Supply Manager→ Site Navigator→ Surveillance→ Chaos Intruder→ Conscripts→ Chaos Device→ Site Director→ O5

MasterCard / Conscripts は独自のギャンブル枠。Chaos Device は Captain 相当で、Very Fine から Site Director へ進む。

権限一覧

この節を権限の正規参照とする。各 Level は下位レベルを含む。Administration は Checkpoints + Intercom = Admin1、ExitGates = Admin2、AlphaWarhead = Admin3。API では vanilla は KeycardAccessLevels.FromVanillaItem(ItemType)、custom は CItemKeycard.AccessLevels から参照できる。
