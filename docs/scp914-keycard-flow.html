SCP-914 Keycard Flow (新仕様ドラフト)

Janitor < Scientist < Zone Manager < Research Coord. < Containment Eng. < Facility Manager < Site Director < O5、軍系は Armory1 < Guard = Chaos Intruder < Armory2 < Security Chief < MTF Private < MTF Operative < MTF Captain = Chaos Device < Site Director < O5 という階段をベースにした概要表。

メモ・前提

共通 O5 ロール 全ての入力に対して、処理の最初に 0.2% 程度で O5 に変換される超低確率ロールが走る想定（床ピックアップ/インベントリ共通）。
研究・管理系階層 Janitor < Scientist < Zone Manager < Research Coordinator < Containment Engineer < Facility Manager < Site Director < O5 の順で上がっていく。
軍・武器系階層 Armory 1 < Guard = Chaos Intruder < Armory 2 < Security Chief < MTF Private < MTF Operative < MTF Captain = Chaos Insurgency Device < Site Director < O5 のイメージ。
O5 / OMNI 的立ち位置 Site Director と同格以上の最終カード。OMNI のようにほぼ全権限を持ち、914 からは上位層やギャンブル用カード（MasterCard / Chaos ルート）経由でのみ到達する。

大まかなフローチャート

研究 / 管理ルート

Janitor→ Scientist→ Zone Manager→ Research Coord.→ Containment Eng.→ Facility Manager→ Site Director→ O5

Fine で基本 1 段階アップ、Very Fine で 1～2 段アップ＋一部 Destroy / ギャンブル（O5 抽選）。

軍 / 武器アクセスルート

Armory 1→ Guard⇄ Chaos Intruder→ Armory 2→ Security Chief→ MTF Private→ MTF Operative→ MTF Captain / Chaos Device→ Site Director→ O5

軍系は武器アクセス段階とセキュリティ段階が混在。Fine で Armory / MTF が順に強化され、Very Fine で Captain / Chaos Device / O5 など上位へ。

Chaos / ギャンブルルート

Quarter⇄ MasterCard⇄ PlayingCard→ Supply Manager→ Site Navigator→ Surveillance→ Chaos Intruder→ Conscripts→ Chaos Device→ O5

MasterCard の Very Fine や Conscripts / Chaos Device の Very Fine から、低～中確率で O5 / Site Director / 高位 Chaos カードへ飛ぶギャンブル枠。
