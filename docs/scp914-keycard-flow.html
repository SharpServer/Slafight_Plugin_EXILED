SCP-914 Keycard Flow

研究系は指定どおり RC < Zone Manager < Scientist の順に整理。SCP:SL公式Wikiの Containment / Security / Administrative Access の考え方に寄せ、武器アクセス I-II-III とSite Director分岐を追加。

共通事故床Pickupは最初に1/12でワイルドカード判定。成功時はSCP-513が1/10、CapybaraMissileが9/10。
O5SCPCBのOMNI相当として最終段。公式WikiでもO5は全権限カード枠として扱われる。
Site DirectorFacility Managerから50%で分岐する横道。権限はFacility Manager相当で、O5ほど万能ではない。
Chaos IntruderChaos側の低位侵入デバイスとして914表に追加。ConscriptsとChaos Insurgency Deviceの間に入る。

主要ルート

研究/管理

Janitor→ Research Coord.→ Zone Manager→ Scientist→ Containment Eng.→ Facility Manager→ Site Director / O5

武器アクセス

Guard→ Armory I→ Armory II→ Armory III→ Captain / O5

Chaos

Conscripts→ Chaos Intruder→ Chaos Device→ Armory III→ O5
